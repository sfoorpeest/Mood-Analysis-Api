# AI Mood Analysis API

A simple backend service that analyzes user emotions from text using AI.

## Tech Stack
- Python
- FastAPI
- HuggingFace Transformers

## Features
- Analyze mood from text
- Return emotion label and confidence score

## API
- POST /api/mood/analyze

## Future Improvements
- Store analysis history in MongoDB
- Emotion classification (sad, stress, happy, angry)
- Mental health advice system
